# cloudflare-bot - บอทตั้งค่า cloudflare

auto config cloudflare

## Installation
```bash
npm install
npm run build
```
## Configuration
rename [`data.example.txt`](data.example.txt) file to `data.txt`
```bash
cp data.example.txt data.txt
```
and edit `data.txt` file

## Execute

```bash
npm start
```
